Welcome to the official **Dinosaur Exploder** news blog! From here on, we'll be sharing development updates, new releases, community highlights, and everything happening around the project.

## About the Project

Dinosaur Exploder is an open-source retro shoot 'em up built with **Java**, **JavaFX**, and **FXGL**. It started as a personal project to explore game development with the Java ecosystem, and quickly grew into something the community started contributing to.

## What to Expect Here

We'll be posting about:

- **New releases** and what changed in each version
- **Contributor spotlights** — highlighting the people who make this possible
- **Development diaries** — behind-the-scenes looks at features in progress
- **Community updates** — events, discussions, and milestones

## Get Involved

The best way to follow along is to:

1. ⭐ **Star the repo** on [GitHub](https://github.com/jvondermarck/dinosaur-exploder)
