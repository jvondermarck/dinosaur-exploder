After months of development and testing, **Dinosaur Exploder v1.0** is officially out! This is the first public stable release, and it marks the moment the project is ready for the world to play — and contribute to.

## What's in v1.0

### Core Gameplay

- Move your spaceship with the **arrow keys**
- **Shoot** incoming dinosaurs with the spacebar
- Use the **shield** (`E` key) to block incoming hits
- Drop a **bomb** (`B` key) to clear the entire screen instantly
- Press `Esc` to pause at any time

### Features

- Multiple enemy waves with increasing difficulty
- Score system with real-time tracking
- Retro pixel art aesthetic with smooth FXGL rendering
- Multilingual UI: **English, Spanish, French, Greek, Chinese**
- Runs on Windows, macOS, and Linux

### Technical Highlights

- Built with **Java 21 + JavaFX + FXGL**
- Clean architecture: entities, systems, and components decoupled
- Open source under the **MIT license**
- CI/CD pipeline via GitHub Actions
